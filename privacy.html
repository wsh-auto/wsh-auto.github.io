Privacy Policy for gg CLI

Last updated: 2026-01-13

Overview

gg is a command-line tool for personal productivity across multiple Google accounts (different domains, owned by the same person). It provides unified access to Google APIs (Calendar, Drive, Docs, YouTube) from a single CLI interface.

Data Collection

This application:

  • Does NOT collect any personal data
  • Does NOT store data on external servers
  • Does NOT share data with third parties
  • Does NOT use analytics or tracking

Data Storage

  • OAuth refresh tokens are stored locally in 1Password (user's own vault)
  • No data is transmitted to any server other than Google's APIs
  • All data remains on the user's local machine

Google API Usage

This app accesses Google APIs solely to:

  • Read and write calendar events
  • Read and write documents, spreadsheets, presentations
  • Read and write files in Google Drive
  • Read contacts
  • Read and manage tasks
  • Access YouTube data

All API access is initiated by the user via CLI commands. No background or automated access occurs.

Data Deletion

Users can revoke access at any time via:

  • Google Account settings: https://myaccount.google.com/permissions
  • Deleting local refresh tokens from 1Password
