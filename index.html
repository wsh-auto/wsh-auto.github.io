gg - Google APIs CLI

A private command-line tool for personal productivity, providing unified access to Google APIs (Calendar, Drive, Docs, Contacts) with multi-account OAuth2 support.

This application is built by and for a single user to manage their own Google accounts. It is not a public service and has no external users.

Features

  • Multi-account OAuth2 authentication
  • Google Calendar management
  • Google Drive access
  • Google Docs operations
  • Google Contacts integration
